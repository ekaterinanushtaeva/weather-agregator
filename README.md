# weather-agregator
Запуск приложения: в idea выбираем edit configuration (правая верхняя часть окна, рядом с кнопками для старта и отладки, выпадающий список, в нём пункт). Затем в открывшемся окне нажимаем на + в левой верхней части (добавить конфигурацию), выбираем в списке Maven:
Working directory: путь к папке weather-agregator
Command line: clean tomcat7:run-war
Нажимаем ок и затем кнопку Run в правой	верхней части окна idea.
Если всё верно настроено, то по адресу http://localhost:9000/ выведется надпись hello.
